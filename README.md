# 2D Character Motion Pack

このリポジトリは、2Dゲーム向けに作成された**キャラクターのモーションアセット集**です。  
ドット絵スタイルで、歩行・攻撃・待機などのアニメーション素材が収録されています。

## 🎮 利用シーン

- 2Dアクションゲーム
- RPGやアドベンチャーゲーム
- チュートリアルやプロトタイプ制作 など

## 📁 収録内容

- `/idle/` ... 待機モーション
- `/walk/` ... 歩行モーション
- `/jump/` ... ジャンプモーション
- `/fall/` ... 落下モーション
- `/attack/` ... 攻撃モーション
- 各フレームは PNG形式（64x64）
